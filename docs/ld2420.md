# Spécifications Techniques : Radar mmWave LD2420

Capteur de présence humaine haute sensibilité utilisant la technologie radar FMCW 24GHz. Contrairement au PIR, il détecte les micro-mouvements (respiration).

!Image du capteur

## 🛠 Caractéristiques Électriques

| Attribut | Valeur |
| :--- | :--- |
| **Tension de fonctionnement** | 3.3V |
| **Consommation moyenne** | ~50 mA |
| **Fréquence** | 24 GHz - 24.25 GHz |
| **Interface de commande** | SPI (pour ESPHome) |
| **Distance de détection** | Jusqu'à 8 mètres |

## 📈 Performances de Mesure

- **Détection de mouvement :** Pour les personnes marchant ou bougeant.
- **Détection de présence :** Pour les personnes immobiles (lecture, sommeil).
- **Précision :** Découpage de la zone en "gates" (fenêtres de distance).
- **Angle :** ±60° horizontal / vertical.

## 🔌 Brochage (Pinout)

| Pin LD2420 | Pin ESP32-C6 | Fonction |
| :--- | :--- | :--- |
| **VCC** | 3.3V | Alimentation |
| **GND** | GND | Masse |
| **SCK** | GPIO 7 | SPI Clock |
| **TX/MISO** | GPIO 5 | SPI MISO |
| **RX/MOSI** | GPIO 6 | SPI MOSI |
| **CS** | GPIO 20 | Chip Select |

## 💡 Notes de Configuration ESPHome

- Le capteur utilise le bus SPI global.
- Il est recommandé de l'utiliser en complément d'un PIR : le PIR pour le déclenchement immédiat (très réactif) et le LD2420 pour maintenir l'écran allumé tant qu'une personne est présente sans bouger.
- La configuration des "gates" peut être ajustée via l'outil de configuration LD2420 d'ESPHome.
