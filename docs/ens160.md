# Spécifications Techniques : Capteur Qualité de l'Air ENS160

Capteur multi-gaz numérique conçu spécifiquement pour le contrôle de la qualité de l'air intérieur.

!Image du capteur

## 🛠 Caractéristiques Électriques

| Attribut | Valeur |
| :--- | :--- |
| **Tension de fonctionnement** | 1.71V ~ 1.98V (Module souvent régulé à 3.3V) |
| **Courant max** | 29 mA (en chauffe) |
| **Interface de commande** | Bus I2C / SPI |
| **Adresse I2C** | 0x53 (par défaut) ou 0x52 |
| **Temps de préchauffage** | < 3 minutes |

## 📈 Performances de Mesure

- **Sorties :** AQI (1-5), TVOC (ppb), eCO2 (ppm).
- **AQI (Air Quality Index) :** Basé sur les standards internationaux.
- **TVOC :** 0 à 65 000 ppb.
- **eCO2 :** 400 à 65 000 ppm.
- **Compensation :** Supporte la compensation de température et d'humidité externe.

## 🔌 Brochage (Pinout)

| Pin | Fonction | Description |
| :--- | :--- | :--- |
| **VCC** | Alimentation | 3.3V (sur module) |
| **GND** | Masse | Référence 0V |
| **SCL** | I2C Clock | Horloge du bus I2C |
| **SDA** | I2C Data | Données du bus I2C |
| **ADDR** | I2C Select | Sélection d'adresse (L=0x53, H=0x52) |
| **INT** | Interrupt | Sortie d'interruption (optionnelle) |

## 💡 Notes de Configuration ESPHome

- Le capteur nécessite un temps de stabilisation pour des mesures de COV précises.
