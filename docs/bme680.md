# Spécifications Techniques : Capteur Environnemental BME680

Capteur 4-en-1 haute précision pour la mesure des gaz, de la pression, de l'humidité et de la température.

!Image du capteur

## 🛠 Caractéristiques Électriques

| Attribut | Valeur |
| :--- | :--- |
| **Tension de fonctionnement** | 1.7V ~ 3.6V |
| **Courant moyen** | 2.1 µA (1 Hz monitoring) |
| **Interface de commande** | Bus I2C / SPI |
| **Adresse I2C** | 0x77 (par défaut) ou 0x76 |
| **Température de service** | -40°C à +85°C |

## 📈 Performances de Mesure

- **Gaz (COV) :** Résistance électrique (Ohm), sensible à l'éthanol et aux COV.
- **Pression :** 300 à 1100 hPa (±0.6 hPa).
- **Humidité :** 0% à 100% RH (±3% RH).
- **Température :** -40°C à 85°C (±1.0°C).

## 🔌 Brochage (Pinout)

| Pin | Fonction | Description |
| :--- | :--- | :--- |
| **VCC** | Alimentation | 3.3V (Typique) |
| **GND** | Masse | Référence 0V |
| **SCL** | I2C Clock | Horloge du bus I2C |
| **SDA** | I2C Data | Données du bus I2C |
| **CSB** | SPI Select | Chip Select (I2C si tiré au VCC) |
| **SDO** | ADDR Select | Sortie données SPI / Adresse I2C LSB |

## 💡 Notes de Configuration ESPHome

- La mesure des gaz nécessite le chauffage de la plaque interne (consommation momentanée plus élevée).
